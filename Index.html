Automated Data Cleaning System

Project Overview

Designed a fully automated SQL-based data cleaning pipeline using stored procedures and event scheduling to regularly clean, format, and maintain data integrity for U.S. household income datasets.

Key Features

  • Created a dynamic procedure to copy data from a staging table and clean it with timestamps for traceability
  • Removed duplicates using ROW_NUMBER() partitioned by id and Timestamp to avoid overlap in automated runs
  • Standardized data entries (e.g., typo fixes, case formatting, type corrections)
  • Scheduled automatic runs every 30 days using a MySQL EVENT
  • Timestamp tracking for each automation cycle to verify execution and data integrity

Impact

Ensured consistent and scalable data quality maintenance with a reusable SQL procedure, reducing manual data cleaning efforts and preserving data history using timestamp tracking.
